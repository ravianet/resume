## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic knowledge of HTML/CSS if you want to customize
- Git installed for version control

### Local Development

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd resume
   ```

2. **View locally**
   Simply open `index.html` in your web browser to view the resume.

3. **Make changes**
   Edit the HTML/CSS to personalize your resume. Update the PDF/Word documents with your latest information.

### Deployment

This website can be deployed to any static hosting platform:

- **GitHub Pages**: Free hosting directly from your repository
- **Vercel**: One-click deployment with automatic SSL
- **Netlify**: Continuous deployment from Git
- **Any web host**: Upload the files via FTP

To deploy to GitHub Pages:
1. Push your code to a GitHub repository
2. Go to Settings → Pages
3. Enable GitHub Pages from the main branch
4. Your resume will be live at `https://<username>.github.io/<repo-name>/`

## 📝 Customization Guide

1. **Update Content**: Modify `index.html` to reflect your latest experience
2. **Replace Assets**: Update `avatar.png`, `mylogo.png`, and `hero_bg.png` with your own images
3. **Refresh Documents**: Update `Resume.pdf` and `.docx` with your most recent resume
4. **Styling**: Edit the CSS in `index.html` to match your personal brand colors

## 🔄 Updating Your Resume

1. Make changes to your Word document (`Resume(09-05-26).docx`)
2. Export a new PDF version
3. Run `extract_resume.py` to update the text content if needed
4. Update the web content in `index.html`
5. Use `push_to_github.bat` to automate the deployment process
